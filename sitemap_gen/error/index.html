Contains error types specific to sitemap operations.

This module defines a comprehensive set of error types that can occur during sitemap creation, modification, and serialization processes. Error types for the sitemap library.

This module defines various error types that can occur during sitemap operations, including XML parsing, date handling, URL parsing, and I/O operations.

The main error type is SitemapError, which encapsulates all possible errors that can occur within the library. This allows for consistent error handling and propagation throughout the codebase.
